# AEautoLauncher (AEselector)

After Effectsのプロジェクトファイル(.aep)のバージョンを自動判別し、適切なバージョンのAfter Effectsを起動するランチャーです。

## 概要

`.aep` ファイルのヘッダーを解析し、作成されたバージョンを特定します。そのバージョンに対応するインストール済みのAfter Effectsを自動的に探し出して起動します。

## 主な機能

- **バージョン自動判別**: AE 5.0 ～ 最新のCCバージョンまで対応。
- **自動起動**: 判別したバージョンがインストールされていれば、そのバージョンで起動します。
- **デバッグモード**: `Ctrl`キーを押しながら起動すると、実際にAEを起動せずに検出されたバージョン情報をダイアログ表示します。
- **軽量**: フォーム画面を持たないコンソールアプリケーションとして動作し、高速に処理を行います。

## 対応バージョン

- After Effects 5.0, 5.5, 6.0, 6.5, 7.0
- CS3, CS4, CS5, CS5.5, CS6
- CC, CC 2014, CC 2015, CC 2017, CC 2018, CC 2019
- CC 2020以降 (自動マッピング対応)

## 使い方

1. ビルド済みの `AEautoLauncher.exe` を適当な場所に配置します。
2. `.aep` ファイルを右クリックし、「プログラムから開く」→「別のプログラムを選択」を選びます。
3. `AEautoLauncher.exe` を選択し、「常にこのアプリを使って .aep ファイルを開く」にチェックを入れます。
4. 以降、`.aep` ファイルをダブルクリックするだけで、適切なバージョンのAEが起動します。

## ビルド環境

- Visual Studio 2019 以降
- .NET Framework 4.7.2 (またはそれ以降)
